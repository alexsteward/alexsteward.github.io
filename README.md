# The Dojo Website

Welcome to my personal project for creating a website for **The Dojo**, a Minecraft PvP community that I lead. This website serves as a platform to showcase our community's features, benefits, and how to join us.

## Project Overview

This project is an exploration of web development, where I aim to learn and implement HTML and CSS to create a functional and appealing website. The Dojo community focuses on refining PvP skills and providing training for players of all levels. 

## Features

- Information about the community and its goals
- List of features and benefits of joining
- Instructions on how to join
- Links to our Discord server

## Technologies Used

- **HTML**: For structuring the content
- **CSS**: For styling the website and making it visually appealing
